## 🚀 Cómo Usar

### 1. Clonar el Repositorio

```bash
git clone https://github.com/tu-usuario/invitacion.git
cd invitacion
```

### 2. Abrir en el Navegador

Simplemente abre `index.html` en tu navegador web:

```bash
# En Windows
start index.html

# En macOS
open index.html

# En Linux
xdg-open index.html
```

O sirve los archivos localmente:

```bash
# Usando Python 3
python -m http.server 8000

# Usando Python 2
python -m SimpleHTTPServer 8000

# Usando Node.js (con live-server)
npx live-server
```

Luego abre `http://localhost:8000` en tu navegador.

### 3. Acceder con tu Carnet

Ingresa tu número de carnet en el campo de autenticación para ver tu invitación personalizada.

## 💡 Personalización

Para modificar los invitados o temas, edita el objeto `GUESTS` en `script.js`:

```javascript
const GUESTS = {
  "carnet": { 
    name: "Nombre", 
    theme: "nombre-tema", 
    confetti: ["#color1", "#color2", "..."] 
  },
  // Agrega más invitados aquí
};
```
